# Deterministic Lookup for Query Handling — Experiment

This repository contains the experiment code for the paper "Deterministic Lookup for Query Handling: Identifying a Cross-Domain Design Pattern."

## Quick Start

python experiment.py

## Expected Output

See Section 5.2 of the paper for expected results.

## Data

- 145 real mineral names from IMA public list (embedded in experiment.py)
- 4,855 synthetic fillers generated programmatically
- 100 error queries generated by Chinese Pinyin input method rules
